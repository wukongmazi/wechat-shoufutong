# 微信电商收付通系统

#### Description
传统的电商平台都是在用户确认收货后再在一定的周期内结算给商户，就是采取平台先收款的方式，但这样不仅给电商平台带来税务的问题，还影响商户的资金安全。有了电商收付通，就不存在这样的问题了。
